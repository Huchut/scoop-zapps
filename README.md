# scoop-zapps

## 仓库设计目标

- 使用scoop代替正常安装，其中包括文件关联，右键菜单功能。
- 安装过程无需手动参与（静默安装）
- 软件仓库定时自动更新(默认每小时)
- 优先采用便携软件，其次绿色软件，最后是安装软件。
- 收录的软件没有弹窗，后台以及绑定主页的流氓行为，有的话就替换成去广告绿色版。
- 安装需要管理员权限的软件时，不需要使用sudo方式安装（请关闭UAC）
